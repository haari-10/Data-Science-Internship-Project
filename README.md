# Data Science Internship Project

### Overview
This project was completed as part of a Data Science Internship and includes:
- COVID-19 Data Analysis
- Loan Prediction using Machine Learning

---

### Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

---

### Project Structure
- Covid -19 Data Analysis (part 1).ipynb → Data cleaning & EDA
- Loan Dataset (part-2).ipynb → Logistic Regression model
- dataset/ → Contains CSV files
- requirements.txt → Required libraries

---

### COVID-19 Data Analysis
- Performed data cleaning and preprocessing  
- Conducted Exploratory Data Analysis (EDA)  
- Used groupby and aggregation techniques  
- Visualized trends using charts  

---

### Loan Prediction (Machine Learning)
- Handled missing values using forward/backward fill  
- Applied one-hot encoding for categorical variables  
- Built Logistic Regression model  
- Evaluated using accuracy score and confusion matrix  

---

### Key Features
- Data preprocessing and cleaning  
- Feature engineering  
- Data visualization  
- Predictive modeling
